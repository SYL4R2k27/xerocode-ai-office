  • Easing: cubic-bezier(0.32, 0.72, 0, 1)
  • Hover: −4px translateY + accent glow
  • Top accent-bar 2-3px на каждой карточке
  • «⌘ NN» numbering префикс везде

Что МОЖЕТ меняться между направлениями

▸ ВИЗУАЛЬНЫЙ ХАРАКТЕР

Mesh-blob / Bento-grid / SOC dashboard / Editorial magazine / Terminal-blueprint — всё разное.

▸ HEADLINE FONT

Каждое направление выбирает свой headline: DM Serif / Instrument Italic / Inter Tight 800 / Space Grotesk / Iosevka mono. Body Inter — общий.

▸ ЦВЕТОВАЯ ГЛУБИНА

Палитра продукта остаётся, но насыщенность и роль меняется. SYLAR может быть пастельным или контрастным.

▸ АНИМАЦИИ

Editorial = scroll-driven type reveal. Bento = hover-lift. SOC = live data. Studio = mesh morph. Каждый — свой signature motion.

SECTION 11
P. 22 · ANIMATIONS
⌘ 11

Animation library

12 паттернов, которые работают на всех 9 направлениях. Дозировка зависит от выбранного направления — но язык движения един.

▸ 01 · TYPE REVEAL

Scroll-driven headline reveal

Большой заголовок зашивается по словам с blur+opacity-mask по мере приближения к центру viewport. Sticky-pinned секция.

Framer Motion useScroll + mask-image · 600-900ms ease

▸ 02 · IMAGE ZOOM

Cinematic image zoom

Изображение в 100vh контейнере scale 1.0 → 1.15 при scroll-progress 0 → 1. Создает «погружение» на ключевые визуалы.

CSS transform + IntersectionObserver · GPU layer

▸ 03 · MAGNETIC CTA

Magnetic cursor-follow

Кнопка «тянется» к курсору в радиусе 80px (translate scaled by 0.3). На hover background меняется на gradient.

requestAnimationFrame + lerp · transform translate3d

▸ 04 · LIQUID BLOB

Liquid mesh morph

Радиальные градиенты медленно дрейфуют (8-12s loop). На mousemove центры тянутся к курсору с задержкой 600ms.

CSS radial-gradient + transform · noise overlay

▸ 05 · STAGGER REVEAL

Cards stagger entrance

Сетка карточек проявляется по диагонали с staggerChildren 60ms. Каждая стартует с y +12px, blur 8px → 0.

Framer staggerChildren · viewport once · easeEmphasis

▸ 06 · STICKY SCRUB

Sticky scroll-scrub

Pin-section на 200vh: при скролле меняется состояние (например, шаги процесса). Текст выводится по фазам, картинка свапается.

position: sticky + scroll-progress + AnimatePresence

▸ 07 · PARALLAX LAYERS

Parallax multi-layer

3 слоя движутся с разными скоростями: фон 0.3x, средний 0.6x, передний 1.0x. Создает глубину в hero.

useScroll + useTransform · transform translate

▸ 08 · COUNTER ANIM

Numbers count-up

При появлении в viewport число анимируется от 0 до целевого значения. Tabular-nums чтоб не дрожало.

requestAnimationFrame · ease-out cubic · 1.2-1.8s

▸ 09 · LIVE FEED

Live data feed

Лог-строки появляются сверху каждые 3-5 секунд. Старые исчезают снизу. Ощущение реальной системы.

setInterval + AnimatePresence · slide+fade

▸ 10 · MARQUEE BAND

Bottomless marquee

Бесконечная горизонтальная лента (логотипы / ключевые слова / цитаты). 30s loop, paused на hover.

CSS @keyframes translateX · duplicated content

▸ 11 · CURSOR TRAIL

Cursor spotlight

Большой radial-gradient следует за курсором (lerp 0.06). Подсвечивает контент под ним. Не утомляет.

requestAnimationFrame + radial-gradient mask

▸ 12 · TYPEWRITER

Terminal typewriter

Текст печатается посимвольно (50-80ms/char). Курсор мигает. Циклом для CLI-секций.

setInterval + slice + cursor blink keyframes

▸ 13 · KEN BURNS

Image Ken Burns

Hero-картинка медленно зумится (1.0 → 1.08) и пансит (translate 2-3%) на протяжении 14-18 секунд. Loop alternate. Hero никогда не «застывает».

CSS @keyframes infinite alternate · transform-origin: center

▸ 14 · SCRUB MASK

Image scrub-clip

Большая картинка раскрывается через clip-path inset на скролле: от 30% → 0%. Чувство «затвора фотоаппарата». Хорошо для product shots.

useScroll + useTransform → clip-path: inset(N%)

▸ 15 · SHADER NOISE

Background noise field

SVG turbulence фильтр поверх gradient hero. Тонкая зернистость, дрожит на 0.3 opacity. Спасает «плоский dark» от пустоты.

SVG feTurbulence · CSS filter: url(#noise) · GPU layer

▸ 16 · MAGNETIC SVG

Cursor node attraction

SVG-узлы (карта, schema) тянутся к курсору с lerp 0.04-0.08. Edge-линии перерисовываются. Идеально для SYLAR · B (Living Map).

requestAnimationFrame · lerp · SVG path d-attr

▸ 17 · TILT 3D

Card 3D tilt

Карточки наклоняются за курсором (rotateX/rotateY ±8°), внутри — глянцевый highlight, который двигается. Stripe-вибрация.

mouseenter/move · transform: perspective(1000px) rotate3d

▸ 18 · NUMBER SCRAMBLE

Hash scramble

Цифры/коды постоянно «перешафливаются» (10 кадров) перед стабилизацией. На каждом ховере — реролл. Чувство «живой системы».

setInterval + random char · settle 600ms · monospace tabular-nums

▸ 19 · GRADIENT TRAVEL

Edge gradient pulse

По линиям-связям бежит светящаяся точка от A к B. Период 3-4с. Создает ощущение «по системе течёт сигнал».

@keyframes left/right · linear-gradient mask · ::after dot

▸ 20 · LENIS SMOOTH

Smooth scroll engine

Lenis заменяет нативный scroll: lerp 0.1, easing exponential. Все scroll-driven эффекты получают «масло». Используется на всех 9.

@studio-freight/lenis · raf integration · 60fps

▸ 21 · IMG SWAP

Picture state swap

Один контейнер — несколько картинок. На скролл-этапе один скрывается (clip + scale 1→0.92), другой проявляется. Sticky-pinned секция.

AnimatePresence · scrollYProgress thresholds · clip-path

▸ 22 · PAGE EXIT

Route exit & enter

Между страницами — мгновенный «занавес» из aurum-полосы (300ms in, 300ms out). Контент следующей страницы уже готов снизу.

AnimatePresence mode="wait" · y/opacity · ease-emphasis

Моя ставка (subjective)

«SYLAR · B — Living Map + XeroCode · A — Editorial Italic + RoleFlow · B — Enterprise Trust»

SYLAR-зонт получает Living Map — интерактивную карту с двумя планетами-продуктами. Это не «ультра-премиум», не «editorial luxury» и не «ещё один SaaS landing». Это аттракцион: человек прилетает на sylar.com и сразу видит карту своих будущих инструментов. Память от такого опыта в 10× выше стандартного hero+features+pricing.

XeroCode остаётся в Editorial Italic — это его душа, и она уже работает на v3.

RoleFlow идёт в Enterprise Trust — это язык CISO/CIO, которые принимают решение о покупке. SOC-Dashboard остаётся как live demo внутри.

Cross-product DNA (раздел 12½) гарантирует что несмотря на радикально разные фронты — все три читаются как «одна семья»: ⌘ marker, Aurum touchpoint, italic emphasis, JetBrains Mono labels, единый ease.
