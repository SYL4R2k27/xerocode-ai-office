Design a modern AI orchestration dashboard web application.

The product is an "AI Operating System" where multiple AI agents collaborate like a team of employees inside a digital office.

The interface must feel alive, interactive, and intelligent — not like a typical dashboard.

---

STYLE:

- Clean minimalistic design (Apple + Material 3 hybrid)
- Soft shadows, glassmorphism elements
- Rounded corners (16–24px)
- Smooth spacing and breathing layout
- High-end SaaS look (Notion + Linear + Apple)
- Subtle gradients and depth
- Elegant typography (Inter / SF Pro style)

---

COLOR SYSTEM:

Primary: #6B2FA6 (deep purple)
Background: #F7F7F9 (light gray)
Surface: white cards
Text primary: #1E1E1E
Secondary text: #6B6B6B

Status colors:
- Working: blue
- Thinking: purple
- Done: green
- Error: red

---

MAIN LAYOUT (single screen app):

1. TOP BAR
- App name: "AI Office"
- Connection status (online/offline)
- Active agents counter
- User profile

---

2. LEFT PANEL (CONTROL PANEL)

- Input field (large, focus element)
  placeholder: "Give a task to your AI team..."

- Buttons:
  - Command
  - Edit
  - Idea

- Recent commands history

---

3. CENTER (OFFICE VIEW)

This is the key feature.

Design a digital office environment where:

- Each AI agent is represented as a character or avatar
- Each agent has a workspace (desk or card)
- Agents visually show states:
  - Thinking (subtle glow / animated dots)
  - Working (typing animation / moving elements)
  - Idle (calm state)
  - Completed (checkmark highlight)

- Agents can:
  - move slightly
  - send messages to each other (animated lines or chat bubbles)
  - show progress bars

- Layout should feel like a team working together

---

4. RIGHT PANEL (TASK GRAPH)

- Visual task tree using nodes and connections
- Each node = task
- Show dependencies
- Color-coded status
- Expand/collapse nodes
- Smooth transitions

---

5. BOTTOM PANEL (LOGS / CHAT)

- Live conversation between agents
- Messages styled like Slack / terminal hybrid
- Show:
  - agent name
  - message
  - timestamp
  - reasoning (optional collapsed view)

---

INTERACTIONS:

- Real-time updates feeling
- Smooth transitions between states
- Hover effects with soft elevation
- Animated transitions (200–300ms)

---

MICRO-INTERACTIONS:

- typing indicator for agents
- subtle pulsing glow when thinking
- task completion animation
- connection lines appearing between agents

---

COMPONENTS TO DESIGN:

- Agent card / avatar
- Task node
- Input panel
- Status badge
- Activity indicator
- Chat message bubble

---

ADDITIONAL SCREENS:

1. Settings page:
- API keys (OpenAI, Claude)
- Local AI toggle
- Model preferences
- Limits indicator

2. Agent configuration:
- Enable/disable agents
- Assign skills

---

UX GOALS:

- Feel like managing a team, not software
- Visual clarity of progress
- Emotional engagement (alive system)
- Minimal cognitive load

---

DO NOT:

- Make it look like a boring admin dashboard
- Avoid tables as primary UI
- Avoid heavy clutter

---

OUTPUT:

- High fidelity UI
- Multiple components
- Clean layout system
- Ready for development handoff
